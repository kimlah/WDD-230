Site Purpose

~Scoots is a website used to convey a 3 day weather forecast in Cozumel, inform the user of tours, what is available to rent (scooters, ATVs and Jeeps) and what the cost will be for a specified number of days.

The webpage will have images that show local tours and vehicles available.

The site will have responsive design techniques that will allow users the ability to view the site well from any device. This will make it easier to rent transportation while in Cozumel, check the weather for the next 3 days and show recommended tours. Contact information will be readily available to allow employees to answer any questions.
